# Exchanging-Any-Kind-Of-Help-and-Unneccesary-Items-For-Free
Developed a platform for free give and take of goods with great consideration for the security of the users. 
(Technologies used: Spring boot, hibernate, thymeleaf, Bootstrap5, Html, Css; Languages used: Java, Javascript;IDE-Spring Tool Suite)
